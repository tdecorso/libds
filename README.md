# Data Structures library in C

## Features
This library currently implements:
  - dynamic arrays
  - stacks (using dynamic arrays)
  - linked lists

## Dependencies
  - <stdlib.h>
  - <stdio.h>
  - <string.h>
  - <assert.h>

## Build instructions
```code
git clone git@github.com:tdecorso/libds.git
cd libds
mkdir build && cd build
cmake ..
make
```
